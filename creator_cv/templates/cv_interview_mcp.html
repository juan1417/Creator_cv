Cómo funciona

Respondes en el navegador; la app fusiona tus respuestas en el JSON del CV y en la revisión Markdown. Las preguntas las prepara un archivo cv-interview.pending.json en tu carpeta context (normalmente desde Cursor con el MCP de archivos, o con el botón de plantilla más abajo).

Cursor, Gemini y MCP

Esta página la sirve Flask y solo lee el JSON en la ruta indicada. No llama a la API de Gemini. La clave GEMINI_API_KEY en .env sirve al servidor MCP creator_cv.mcp.gemini_server en Cursor (herramienta gemini_generate). Eso no crea solo el pending.json: hace falta que la IA o tú escribáis el archivo en context (p. ej. con Filesystem MCP), o usar el botón que copia la plantilla con Python (sin lanzar MCP).
